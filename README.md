# gitActionDemo

# steps

```
1. select node.js configuration file.
2. create github token and set in secret & variable's action.
3. set read & write workflow permissions and save it.
4. add new self-hosted runner.
5. run one by one all commands of runner in server machine.
6. commit & push your changes and run workflow in git action.
7. after complete workflow's job check changes in server machine.
```
